# 全公司钱包设计文档

> 日期：2026-04-20

---

## 1. 概述

新增「全公司钱包」模块，供公司管理员查看和管理公司下的两个钱包：**余额钱包**与**应用钱包**。两个钱包之间可以通过「划转」操作双向转账。

本模块独立于现有公司钱包（`/finance/my-wallet`），不做替换，待评审通过后再决定是否替换。

### 核心概念

- **公司 : 钱包 = 1 : N**，固定为 1 个余额钱包 + 1 个应用钱包
- **余额钱包**：用于追加投资、释放股份
- **应用钱包**：用于企业转单接单（专款专用）
- 两个钱包均支持 USDT / PEA 双币种

## 4. 划转弹窗

点击页面右上角「划转」按钮触发，两步 Modal 流程。

### Step 1 — 填写信息

| 字段 | 控件 | 说明 |
|------|------|------|
| 转出钱包 | Radio（余额钱包 / 应用钱包） | 选择后自动显示当前余额 |
| 转入钱包 | 只读文本 | 自动填充为另一个钱包，显示当前余额 |
| 币种 | Radio（USDT / PEA） | 切换后余额数字跟随变化 |
| 划转金额 | 数字输入框 | 必填 |
| 备注 | 文本输入框 | 可选 |

转出/转入钱包下方均显示对应币种的实时余额，方便用户确认。

### Step 2 — MFA 验证

- 显示划转摘要：转出钱包 → 转入钱包、金额、币种
- 6 位验证码输入
- 确认按钮

MFA 验证方式与集团钱包充值/转出一致。

## 5. 详情弹窗

点击流水记录操作列「详情」按钮触发，使用 `Descriptions` 组件纵向排列，样式与投资审批详情弹窗一致（宽度 640px）。

### 5.1 共性字段（所有订单类型）

流水号、订单时间、钱包类型、订单类型、金额、币种、状态

### 5.2 按订单类型展示特殊字段

**划转：**
- 转出钱包、转入钱包、备注

**追加投资：**（复用投资审批数据结构）
- 企业名称、企业ID
- 企业主昵称、企业主ID、企业主用户名
- 追加投资总金额、本公司占股比例
- 需投资货币单位、需投资金额

**释放股份：**（复用投资审批数据结构）
- 企业名称、企业ID
- 企业主昵称、企业主ID、企业主用户名
- 释放股份比例、总股本预估金额
- 释放股份货币单位、释放股份金额

**转单扣款：**
- 佣金订单号（`commissionOrderId`）
- 「查看佣金订单」跳转链接 → `/commission`

## 6. 数据结构

### 6.1 流水记录

```typescript
type WalletType = 'balance' | 'app';
type OrderType = 'transfer' | 'additional_investment' | 'share_release' | 'order_deduction';
type FlowStatus = 'success' | 'pending' | 'failed';

interface WalletFlowRecord {
  id: string;
  flowId: string;
  createdAt: string;
  walletType: WalletType;
  orderType: OrderType;
  amount: number;
  currency: 'USDT' | 'PEA';
  status: FlowStatus;

  // 划转特有
  fromWallet?: WalletType;
  toWallet?: WalletType;
  remark?: string;

  // 追加投资特有（复用投资审批字段）
  sourceCompanyId?: string;
  sourceCompanyName?: string;
  sourceOwnerId?: string;
  sourceOwnerUsername?: string;
  sourceOwnerNickname?: string;
  totalInvestAmount?: number;
  shareRatio?: number;
  investCurrency?: string;
  investAmount?: number;

  // 释放股份特有（复用投资审批字段）
  releaseRatio?: number;
  totalShareValue?: number;
  releaseCurrency?: string;
  releaseAmount?: number;

  // 转单扣款特有
  commissionOrderId?: string;
}
```

### 6.2 钱包余额

```typescript
interface WalletBalance {
  walletType: WalletType;
  walletName: string;       // '余额钱包' | '应用钱包'
  usdtBalance: number;
  peaBalance: number;
}
```

## 9. 技术要点

- 框架：Umi 4 + React 18 + Ant Design 5 + TypeScript
- 状态管理：React hooks（useState / useMemo / useCallback）
- 样式规范：遵循 `src/styles/design-spec.md`
  - 主色 `#1677ff`
  - 卡片圆角 12px、标准阴影
  - 金额黑色 `#141414`，不用红绿色
  - 表格斑马纹
  - Radio.Button 蓝色主题
- 复用组件：`TableToolbar`
- 路由参数：无（单页，不使用 Tab）
