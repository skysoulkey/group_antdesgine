## 一、概述

商户管理平台需要在关键业务事件发生时，向公司管理员发送消息通知。通知覆盖 **小程序**、**邮箱**、**站内** 三个渠道，确保管理员能及时处理待办事项。

其中「持股企业追加投资」和「持股企业释放股份」两类通知，小程序渠道需要支持 **Inline Keyboard 操作按钮**（同意 / 拒绝），管理员可直接在小程序内完成审批操作。审批的完整状态流转、自动审批规则、Web 端审批页面等详见《公司审批流程设计文档》。

## 二、通知对象

- 默认在平台创建公司管理员时，进行消息通知配置地址，包括公司管理员的 **小程序联系地址**、**邮箱地址**
- 通知对象在「通知管理 → 通知配置」中维护（增删改查）

## 三、通知路径

### 3.1 小程序通知

使用小程序通知（通过系统平台进行管理维护）：

- 小程序的模版样式
- 小程序通知内容
- 小程序通知对象配置
- **审批类通知附带 Inline Keyboard 操作按钮**（参考 Telegram Bot Inline Keyboard）

### 3.2 邮箱通知

使用系统邮箱进行通知（通过系统平台进行管理维护）：

- 邮箱通知模版样式
- 邮箱通知内容
- 邮箱通知对象配置

### 3.3 站内通知

平台内消息中心推送，用户登录后可在通知列表查看：

- 站内通知模版内容
- 阅读状态追踪（已读 / 未读）

### 3.4 API 集成

商户管理平台创建账户时，需要调用系统平台的配置 API 来进行通知对象的增删改查。

## 五、小程序审批按钮

> 审批的完整交互流程、状态流转、边界情况处理详见《公司审批流程设计文档》第 7 节。本节仅描述通知层面的按钮展示规范。

### 5.1 适用通知类型

仅以下两种通知类型在小程序渠道附带操作按钮：

- 持股企业追加投资
- 持股企业释放股份

### 5.2 按钮样式（参考 Telegram Bot Inline Keyboard）

通知消息底部附带两个 Inline Keyboard 按钮：

```
┌─────────────────────────────────────────┐
│ 【nova星球】通知                          │
│ 您管辖的企业有一条待处理事项，              │
│ 请10分钟内处理～                          │
│                                         │
│  ┌──────────┐  ┌──────────┐            │
│  │ ✅ 同意   │  │ ❌ 拒绝   │            │
│  └──────────┘  └──────────┘            │
└─────────────────────────────────────────┘
```

### 5.3 按钮状态变化

| 操作 | 按钮变化 |
|------|---------|
| 点击「✅ 同意」 | 按钮文字变为「已同意 ✓」，按钮置灰不可再点 |
| 点击「❌ 拒绝」 | 按钮文字变为「已拒绝 ✗」，按钮置灰不可再点 |
| 超时未操作 | 按钮自动移除，消息追加「⏰ 已超时」 |

## 六、页面功能

### 6.1 通知记录 Tab

- 按通知方式筛选（小程序 / 邮件 / 站内）：≤6 项，使用 Radio.Button 平铺
- 按通知类型筛选（10 种类型）：>6 项，使用 Select 下拉
- 按通知对象搜索：Input 文本搜索
- 查看通知详情
- **操作状态列**：审批类通知显示操作状态（待操作 / 已同意 / 已拒绝 / 已超时），非审批类显示「—」

### 6.2 通知配置 Tab

- 查看当前通知对象列表（小程序用户名、邮箱）
- 编辑通知对象（新增 / 修改 / 删除）
- 按渠道开关通知（小程序通知、邮件通知独立开关）

### 6.3 通知类型汇总

| 序号 | 通知类型 | 触发时机 | 小程序操作按钮 |
|------|---------|---------|--------------|
| 1 | 集团下拨 | 集团向公司下拨资金时 | 无 |
| 2 | 集团调回 | 集团从公司调回资金时 | 无 |
| 3 | 持股企业追加投资 | 持股企业发起追加投资审批时 | ✅ 同意 / ❌ 拒绝 |
| 4 | 持股企业释放股份 | 持股企业发起股份释放审批时 | ✅ 同意 / ❌ 拒绝 |
| 5 | 企业解散 | 管辖企业解散时 | 无 |
| 6 | 新增订阅企业 | 新企业完成订阅时 | 无 |
| 7 | 企业认证过期 | 下辖企业认证到期时 | 无 |
| 8 | 企业续费成功 | 下辖企业完成续费时 | 无 |
| 9 | 余额不足 | 因公司余额不足导致追加投资或股份转让失败时 | 无 |
| 10 | 审批操作结果 | 审批操作完成 / 超时后自动发送 | 无 |
