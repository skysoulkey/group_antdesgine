> 本文档定义各仪表盘指标的统计口径与计算规则，所有前端展示的数值须与此规范一致。

## 三、通用规则

| 规则 | 说明 |
|-----|------|
| 货币切换 | 切换货币单位（USDT / PEA）后，页面所有模块的数据同步切换至对应货币单位 |

## 四、企业详情 — 牛牛红包（Tab）

> 路径：`/enterprise/detail/:id?tab=niuniuRedpacket`
> 性质：**仅展示数据**，不做前端计算。表中所有金额字段直接展示后端返回值。
> 返佣链路：庄家发起红包 → 下级用户领取 → 系统按比例返佣给该下级的「上级」（推荐人）。

### 4.1 主表字段

| 字段 | 数据来源/口径 |
|------|-------------|
| 发起时间 | 后端返回 |
| 订单ID | 后端返回 |
| 订单状态 | 后端返回，仅二值：已开奖 / 未开奖 |
| 底注 | 后端返回 |
| 红包数量 | 后端返回 |
| 玩法 | 后端返回，仅二值：庄比 / 通比 |
| 赔率 | 后端返回，枚举三值：平倍 / 高倍 / 超倍 |
| 有效时长 | 后端返回，枚举二值：30 秒 / 60 秒。前端展示时附「秒」单位 |
| 赔付金额 | 后端返回（不做前端计算） |
| 抽水金额 | 后端返回（不做前端计算） |
| 返佣金额 | 后端返回（不做前端计算） |
| 庄家昵称 | 后端返回 |
| 庄家ID | 后端返回 |
| 完成时间 | 后端返回；订单状态为「未开奖」时显示 `—` |

### 4.2 订单详情弹窗（领取记录表）

| 字段 | 数据来源/口径 |
|------|-------------|
| 领取时间 | 后端返回 |
| 领取订单号 | 后端返回 |
| 用户昵称 | 后端返回 |
| 用户ID | 后端返回 |
| 用户IP | 后端返回 |
| 领取金额 | 后端返回 |
| 牌面 | 后端返回（牛牛 / 牛一 / 牛八 等文案） |
| 本局盈亏 | 后端返回（不做前端计算；统一黑色文本，不分正负色） |
| 到账 | 后端返回 |
| 抽水 | 后端返回 |

### 4.3 返佣详情弹窗

| 字段 | 数据来源/口径 |
|------|-------------|
| 创建时间 | 后端返回 |
| 返佣任务ID | 后端返回 |
| 用户昵称 | 实际领取红包的下级；展示格式 `名字（庄）` 或 `名字（闲）`，括号内为该用户在本局的角色 |
| 用户ID | 同上 |
| 上级昵称 | 返佣用户的推荐人 |
| 上级ID | 同上 |
| 返佣金额 | 后端返回 |

## 五、企业详情 — 应用费用（Tab）

> 路径：`/enterprise/detail/:id?tab=appFee`
> 性质：仅展示数据，金额字段精确到小数点后 2 位。

### 5.1 主表字段

| 字段 | 数据来源/口径 |
|------|-------------|
| 创建时间 | 后端返回 |
| 订单编号 | 后端返回 |
| 订单类型 | 后端返回，仅二值：应用结算 / 主动分红 |
| 企业盈利总额 | 后端返回（不做前端计算），精确到小数点后 2 位 |
| 应用费用总额 | 后端返回（不做前端计算），精确到小数点后 2 位 |
| 企业盈利明细 | 三项枚举：东方彩票、七星百家乐、企业红包；前端一行展示「东方彩票 X / 七星百家乐 X / 企业红包 X」 |
| 应用费用明细 | 同上结构 |
| 账单日 | 后端返回，格式 `YYYY-MM-DD` |
| 账单月 | 后端返回，格式 `YYYY-MM` |
