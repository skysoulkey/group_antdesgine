# 集团公司管理系统 · 数据口径规范

> 本文档定义各仪表盘指标的统计口径与计算规则，所有前端展示的数值须与此规范一致。

---

## 一、集团仪表盘（集团管理员视角）

### 1.1 KPI 指标卡

| 指标名称 | 计算口径 | 备注 |
|---------|---------|------|
| 集团余额 | — | 集团钱包余额 |
| 公司资产 | — | 暂无注释 |
| 企业资产 | — | 暂无注释 |
| 持股估值 | 公司持有企业股份估值之和 | |
| 东方彩票盈亏 | 公司东方彩票盈亏之和 | |
| 股份收益 | — | 暂无注释 |
| 税费收益 | 公司收到的股份交易税费及分红税之和 | |
| 集团资金下拨 | 集团转账给公司账户金额之和 | |
| 集团资金调回 | 集团从公司账户转回金额之和 | |
| 公司（家） | — | 暂无注释 |
| 参股公司（家） | 本集团下参股公司数 | |
| 企业（家） | 本集团下完成认证的企业总数 | |
| 成员（个） | 本集团下所有企业成员数 | |

### 1.2 折线图卡片

> 横坐标统一按「天」维度，默认展示最近 31 天数据。

| 图表名称 | 统计说明 |
|---------|---------|
| 集团余额 | 展示最近31天集团钱包余额变化趋势 |
| 公司资产 | 展示最近31天集团下所有公司总资产合计的变化趋势 |
| 企业总资产 | 展示最近31天集团下所有企业总资产合计的变化趋势 |
| 持股估值 | 展示最近31天公司持有企业股份估值之和的变化趋势 |
| 企业盈亏 | 展示最近31天集团下所有企业盈亏总和的变化趋势 |
| 企业流水 | 展示最近31天集团下所有企业游戏流水合计的变化趋势 |
| 企业 | 展示最近31天本集团下完成认证的企业数量变化趋势 |
| 企业成员 | 展示最近31天本集团下所有企业成员总数的变化趋势 |

### 1.3 收益贡献排行榜

| 字段 | 计算口径 |
|-----|---------|
| 排名规则 | 默认按照【公司累计盈利】倒序排列 |
| 累计给公司盈利 | 股份盈亏 + 税费收益 + 企业贡献佣金 之和 |

## 二、公司仪表盘（公司管理员视角）

### 2.1 KPI 指标卡

| 指标名称 | 计算口径 | 备注 |
|---------|---------|------|
| 公司总资产 | 企业余额及应用余额之和 | |
| 持股估值 | 公司当前持有企业资产之和 | |
| 股份盈亏 | — | 暂无注释 |
| 税费收益 | 股份交易税费及分红税之和 | |
| 东方彩票盈亏 | 东方彩票订单 - 订单赔付 | |
| 佣金收益 | — | 暂无注释 |
| 佣金支出 | 本公司支出佣金之和 | |
| 集团资金下拨 | — | 暂无注释 |
| 集团资金调回 | 集团从公司账户转出金额之和 | |
| 认证企业（家） | 本公司下完成认证的企业总数 | |
| 成员总数（个） | — | 暂无注释 |
| 参与成员（个） | 参与投注或红包游戏的成员 | |
| 活跃成员（个） | 本日存在登录记录 | |

### 2.2 折线图卡片

> 横坐标统一按「天」维度，默认展示最近 31 天数据。

| 图表名称 | 统计说明 |
|---------|---------|
| 公司总资产 | 展示最近31天公司总资产（企业余额及应用余额之和）变化趋势 |
| 公司盈亏 | 展示最近31天公司综合盈亏变化趋势 |
| 企业总资产 | 展示最近31天本公司下所有企业总资产合计的变化趋势 |
| 认证企业 | 本公司下认证企业数量 |
| 成员总数 | 展示最近31天本公司注册成员总数的变化趋势 |
| 参与成员 | 本日参与东方彩票游戏的成员之和 |

### 2.3 收益贡献排行榜

| 字段 | 计算口径 |
|-----|---------|
| 排名规则 | 默认按照【累计给公司盈利】倒序排列 |
| 累计给公司盈利 | 股份盈亏 + 税费收益 + 企业贡献佣金 之和 |

## 三、通用规则

| 规则 | 说明 |
|-----|------|
| 货币切换 | 切换货币单位（USDT / PEA）后，页面所有模块的数据同步切换至对应货币单位 |
| 数据时效 | 上方模块统计按照页面顶部「数据更新时间」取最新快照数据 |
| 折线图周期 | 横坐标按「天」维度，默认展示最近 31 天 |
