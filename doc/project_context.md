# 项目背景

## 产品定位

**IM 商户管理平台**（原名"集团公司管理系统"），是一款面向 IM 平台商户体系的 PC 端管理后台，用于集团、公司、企业三级组织架构下的资产管理、订单监控、用户管理和财务运营。

## 核心业务

| 模块 | 说明 |
|------|------|
| 仪表盘 | 集团/公司维度的 KPI 卡片、数据走势折线图、收益贡献排行榜 |
| 公司管理 | 公司列表、公司详情（概览/转账/持股）、收益统计、税费管理 |
| 企业管理 | 企业列表、企业详情（概览/资产/订单/成员）、企业邀请、税费 |
| 财务中心 | 集团钱包、公司钱包、我的钱包、资金下拨/调回、收益明细 |
| 公司订单 | 东方彩票订单、佣金订单 |
| 系统设置 | 用户管理（分权）、角色管理、操作日志、通知管理、个人中心 |

## 用户角色

| 角色 | 权限范围 | 入口 |
|------|---------|------|
| 集团管理员 | 全局数据、所有公司管理、资金下拨/调回、用户管理 | 集团仪表盘 → 各模块 |
| 公司管理员 | 本公司数据、企业管理、订单监控、公司钱包 | 公司仪表盘 → 各模块 |
| 只读用户 | 仅查看权限，无操作按钮 | 根据分配菜单 |

## 设计风格

- **定位**：商务、大气、简洁
- **色系**：Ant Design 标准蓝（`#1677ff`）
- **卡片**：纯文字，无图标色块，同行等高
- **图表**：折线图用圆形标记 + 水平虚线网格
- 详细规范见 `src/styles/design-spec.md`

## 数据状态

当前为 **Mock 阶段**，所有数据均为前端静态模拟，尚未对接后端 API。
